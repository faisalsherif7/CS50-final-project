RATIB is designed to help users easily and accurately calculate and keep track of zakat payments on their salary, based on Shaykh Ibn Baz's fatwa on how to calculate zakat on monthly salaries. Please refer to the fatwa below for details.

To get started, users must first set a nisab amount.

The dashboard tab allows users to add savings, which are automatically calculated and reflected in a table on the same page. If users need to modify a savings amount, they can simply delete the entry and add another one with the desired amount and date using the form provided. The savings table will accurately reflect all savings entries made.

Once a user inputs a zakat amount, RATIB will note and track the amount accordingly based on whether it reaches the nisab or not. If the amount reaches the nisab, it is tracked for the following hijri year and displayed in a zakat due table when the payment becomes due. Users can then declare the amount as paid, and it will be recorded.

The history tab contains information about past paid zakat entries.
